Myanmar CRM Localization

Myanmar address fields for CRM leads and opportunities

Extends CRM leads/opportunities with Myanmar administrative hierarchy fields using official MIMU P-codes. Seamlessly integrates with your sales pipeline workflow.

CRM Integration Features

  • Lead/Opportunity Forms — Myanmar fields in Extra Information tab
  • P-code Auto-fill — Enter P-code, all fields populate
  • Territory Management — Geographic lead assignment
  • Pipeline Integration — Myanmar addresses in kanban view
  • Sales Analytics — Regional opportunity reporting
  • Smart Validation — MIMU database verification

Lead/Opportunity Form

Myanmar address fields appear in Extra Information tab when country is set to Myanmar

Pipeline Integration

Myanmar addresses display correctly in pipeline views for better territory management

Use Cases

  • Sales Teams — Accurate lead location tracking
  • Territory Management — Geographic lead assignment
  • Market Analysis — Regional opportunity reporting
  • Customer Profiling — Detailed address information
  • Sales Reporting — Geographic analysis and statistics
  • Lead Qualification — Location-based scoring

Installation

  1. Install l10n_mm_pcode module first (required dependency)
  2. Place l10n_mm_crm in your Odoo addons path
  3. Update apps list: Apps → Update Apps List
  4. Search for "Myanmar CRM Localization"
  5. Click Install

Myanmar address fields appear automatically in CRM forms.

Requirements

  • Odoo 18.0+
  • CRM module
  • l10n_mm_pcode module
