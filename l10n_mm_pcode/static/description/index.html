Myanmar Administrative Localization

Official MIMU P-code system for Myanmar addresses

Complete administrative hierarchy with State/Region → District → Township → Town → Ward/Village Tract using official Myanmar Information Management Unit (MIMU) Place Codes.

Key Features

  • 18 States/Regions with official MIMU codes
  • 88 Districts including SAZ/SAD
  • 358 Townships mapped hierarchically
  • 536 Towns for urban centers
  • 65,000+ Wards/Village Tracts with unique P-codes
  • P-code Auto-fill — Enter P-code, all fields populate
  • Cascading Selection — Ward → Township → District → State
  • Smart Search — Find by name, P-code, or location
  • Integrated Address Format — Myanmar-standard printing

Installation

  1. Place l10n_mm_pcode in your Odoo addons path
  2. Restart Odoo server (if needed)
  3. Update apps list: Apps → Update Apps List
  4. Search for "Myanmar Administrative Localization"
  5. Click Install

No additional configuration required. Myanmar address fields appear automatically.
