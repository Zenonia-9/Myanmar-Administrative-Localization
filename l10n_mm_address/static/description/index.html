Myanmar Address Localization

Complete Myanmar administrative hierarchy with MIMU P-code & Postal Code integration

Full administrative hierarchy with State/Region → District → Township → Town → Ward/Village Tract using official MIMU P-codes, plus 900+ postal codes and township-level map coordinates.

Key Features

  • 18 States/Regions with official MIMU codes
  • 86 Districts including SAZ/SAD
  • 358 Townships with coordinates and postal codes
  • 800+ Towns for urban centers
  • 16,000+ Wards/Village Tracts with unique P-codes
  • 900+ Postal Codes linked to townships
  • P-code Auto-fill — Enter P-code, all fields populate
  • Cascading Selection — Ward → Township → District → State
  • Postal Code Filtering — Filtered by selected township
  • Map View Support — Township-level coordinates
  • Smart Search — Find by name, P-code, or location
  • Integrated Address Format — Myanmar-standard printing

P-code Auto-fill

Enter a MIMU P-code (e.g., MMR017024040) and watch all address fields populate automatically.

Myanmar Address Fields

Myanmar-specific fields appear automatically when Myanmar is selected as country.

Hierarchical Ward Selection

Map View

Township-level centroid coordinates integrated with Odoo's map view.

Kanban View Integration

Myanmar addresses display correctly in kanban and list views.

Installation

  1. Place l10n_mm_address in your Odoo addons path
  2. Restart Odoo server (if needed)
  3. Update Apps List
  4. Search for "Myanmar - Address Localization"
  5. Click Install

No additional configuration required. Myanmar address fields appear automatically.
