📊 Address Hierarchy

State/Region (res.country.state) └── District (res.district) └── Township (res.township) ├── Town (res.town) │ └── Ward/Village Tract (res.ward) └── Ward/Village Tract (res.ward)

🔢 P-Code Structure

MIMU P-codes are hierarchical 15-character codes:

  • MMR - Country code (Myanmar)
  • 001-018 - State/Region code
  • D001-D333 - District code (or S/SAD for special zones)
  • 001-358 - Township code
  • 0001-9999 - Ward/Village Tract code

Example: MMR017024040 = Myanmar → Ayeyarwady (017) → Pyapon District (024) → Ward/Village Tract (040)

🚀 Usage

Address Entry

When country is set to Myanmar, the following fields appear:

  • District - Filtered by selected State
  • Township - Filtered by selected District
  • Town - Filtered by selected Township
  • Ward - Filtered by selected Town or Township
  • P-Code - Auto-fills from ward selection or manual entry
  • Postal Code - Filtered by selected Township

P-Code Auto-fill

Simply enter a valid MIMU P-code in the P-Code field, and the system will automatically populate:

  • Ward/Village Tract
  • Town (if applicable)
  • Township
  • District
  • State/Region
  • Postal Code

Configuration Access

Navigate to: Contacts → Configuration → Myanmar Localization

  • Districts
  • Townships
  • Towns
  • Wards
  • Post Codes

📋 Technical Details

Models

  • res.district - Districts/SAZ/SAD
  • res.township - Townships with coordinates
  • res.town - Towns
  • res.ward - Wards and Village Tracts
  • res.zip - Post codes
  • res.country - Enhanced with enforce_townships field

Data Sources

This module is built upon data from multiple authoritative sources. We extend our sincere gratitude to:

  • MIMU (Myanmar Information Management Unit) - Official P-codes and administrative boundaries
    https://themimu.info/mm/place-codes
  • Myanmar Post - Official postal code data
    https://myanmarpost.com.mm/postcode
  • Myanmars.net - Postal code reference
    https://www.myanmars.net/postal-code/naypyitaw.html
  • MMR Postcode Base - Comprehensive postal code database
    https://mmr.postcodebase.com/
  • Dr. Nyein Chan's P-Code Database - P-code reference and validation
    https://pcode.drnyeinchan.com/

All data is loaded via CSV files in the data/ folder. Coordinates represent township-level centroids.

Security

  • Read access for all users (base.group_user)
  • Write access for partner managers (base.group_partner_manager)

🏷️ District Types

  • District Regular administrative district
  • SAZ Self-Administered Zone
  • SAD Self-Administered Division

🏘️ Ward Types

  • Ward Urban administrative ward (4-digit P-code suffix)
  • Village Tract Rural administrative division (3-digit P-code suffix)
