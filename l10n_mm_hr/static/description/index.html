Myanmar HR Localization

Myanmar address fields for employee private addresses

Extends employee private addresses with Myanmar administrative hierarchy fields using official MIMU P-codes. Seamlessly integrates with HR management workflow.

HR Integration Features

  • Employee Forms — Myanmar fields in Private Information tab
  • P-code Auto-fill — Enter P-code, all fields populate
  • HR Security — Respects HR access rights
  • Private Addresses — Employee home address management
  • HR Analytics — Employee geographic distribution
  • Payroll Integration — Location-based allowances

Use Cases

  • HR Management — Accurate employee home addresses
  • Payroll — Location-based allowances
  • Emergency Contact — Detailed address information
  • Compliance — Government reporting requirements
  • Analytics — Employee geographic distribution
  • Privacy — Secure private address management

Security & Privacy

  • HR Access Control — Fields restricted to hr.group_hr_user
  • Private Information Protection — Follows Odoo HR security model
  • Data Compliance — Meets data protection requirements
  • Audit Trail — Standard Odoo logging for all changes

Installation

  1. Install l10n_mm_pcode module first (required dependency)
  2. Place l10n_mm_hr in your Odoo addons path
  3. Update apps list: Apps → Update Apps List
  4. Search for "Myanmar HR Localization"
  5. Click Install

Myanmar address fields appear automatically in employee forms.

Requirements

  • Odoo 18.0+
  • HR module
  • l10n_mm_pcode module
